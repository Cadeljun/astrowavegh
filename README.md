# AstroWave Ghana

## Ghana's Next-Generation Entertainment Brand

AstroWave is a creative entertainment brand based in Accra, Ghana, focused on delivering immersive events, dynamic nightlife experiences, and professional talent management. Our mission is to push the boundaries of creativity and provide a platform for Ghana's most talented artists.

### Key Features

*   **Event Management:** A comprehensive system for creating, promoting, and managing events.
*   **Talent Management:** A dedicated portal for showcasing and managing creative talent.
*   **Admin Dashboard:** A powerful and intuitive dashboard for managing all aspects of the platform.
*   **AI-Powered Recommendations:** Leveraging Genkit to provide personalized user experiences.
*   **Content Management System (CMS):** A flexible CMS for managing website content.

### Technologies

*   **Frontend:** Next.js, React, TypeScript
*   **Backend:** Firebase (Firestore, Authentication, Hosting)
*   **AI:** Genkit
*   **Styling:** Tailwind CSS

### Getting Started

To get started with the AstroWave project, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/astrowave.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```
