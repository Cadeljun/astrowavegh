# **App Name**: AstroWave

## Core Features:

- AI Cultural Vibe Navigator: A generative AI tool that curates personalized entertainment and music recommendations based on the user's current mood and vibes.
- Cinematic Ecosystem Hub: Interactive landing interface showcasing active divisions like AstroWave Events and teaser displays for upcoming Record and Care sectors.
- Dynamic Talent Portfolio: Real-time management roster displaying bios, roles, and social presence for artists and DJs, synchronized via cloud database services.
- Integrated Event Registry: Comprehensive listing for experiences like 'Mask Mirage', featuring filtering capabilities and event detail cards.
- Lead Capture Pipeline: Professional inquiry management system that saves partnership and booking details directly to Firestore for the leadership team.
- Future Division Waitlist: Secure sign-up functionality for upcoming subdivisions, allowing fans to join the inner circle for AstroWave Records and Cares.
- Glassmorphism Performance Dash: High-performance visual UI utilizing Frosted-Glass effects and neon accents to provide an immersive browsing experience.

## Style Guidelines:

- The primary color is an energetic Radiant Gold (#FFD166), chosen for its vibrant African creative flair and luxury cinematic feel.
- The background is an Deep Obsidian shade (#0D0D0A), derived from a desaturated gold hue to maintain visual depth and dark mode excellence.
- The accent color is a Sunset Vermillion (#FF5400), an analogous companion providing heat and contrast for interactive elements.
- Headlines use 'Bebas Neue' (Sans-Serif) for high-impact cinematic displays; body and UI text use 'Outfit' (Sans-Serif) for modern readability. Note: currently only Google Fonts are supported.
- Use Lucide React icons with thin geometric strokes and occasional neon gold glows to denote active movement divisions.
- Asymmetric, grid-breaking sections with overlapping elements and diagonal dividers create a forward-thinking cultural aesthetic.
- Fluid section entries and neon pulsing effects on CTA buttons utilize Framer Motion for high-fidelity interactive feedback.
